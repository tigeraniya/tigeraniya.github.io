2. Business Requirements¶

2.1. Login/Signup¶

  1. User will be able to login to system using Callfixie specific cred (Rel 1.0) or Facebook/google/3rd Party Website creds (Rel 2.0).
  2. User will not be required to enter details until he raises a job/confirms it. (Rel 1.0)

2.2. Notifications¶

2.2.1. User¶

  1. New Quotes for posted Jobs. (Rel 1.0)
  2. Ques Ans responses. (Rel 1.0)
  3. Payment events. (Rel 1.0)

2.2.2. Tradie¶

  1. New Jobs Suitable for given Profile and Location. (Rel 1.0)
  2. Ques Ans responses (Rel 2.0)
  3. payment events (Rel 1.0)

2.3. Job Posting¶

  1. Location Information (Rel 1.0)
  2. Category [ plumber, gardener ] (Rel 1.0)
  3. Price Range (Rel 1.0)
  4. Time Range (Rel 2.0)
  5. Tradie Eligibility (Rel 2.0)
  6. Viewability , making a job private ( Rel 2.0 )
  7. User Account should provide credit card details before accepting a Job (Rel 1.0).
  8. Questions Answers on Job ( Rel 2.0)
  9. user should be able to cancel/update Job until it have received a Quote . if a quote is received it will be obsoleted. if Job is approved its Critical information cannot be updated. Job can be cancelled at any point of time. Job cancelled after tradie have started working on it will go to dispute resolution department and refunds will be done by administration .(Rel 1.0)
  10. Quotes on a job can be seen by other tradies. ( Rel 1.0)
  11. user can see all quotes on Jobs. (Rel 1.0)
  12. Attachments ( images Rel 1.0 )( audio, Video Rel 2.0)
  13. Contract-Type ( Rel 1.0 )

2.4. Quotes¶

  1. Description (Rel 1.0)
  2. Attachments ( images Rel 1.0)( Audio,Video Rel 2.0)
  3. Tradie should be eligible by criterias set by Job Owner to quote. ( Rel 2.0 )
  4. Tradies can view jobs they are not eligible for. ( Rel 1.0 )

2.5. Profiles¶

  1. Can be deactivated by User. ( Rel 1.0 )

2.5.1. User¶

  1. profile will not be visible to public (Rel 1.0)

2.5.2. Tradie¶

  1. New Profiles need to be approved by Customer care ( Rel 1.0)
  2. Profile will be skill,Location specific ( Rel 1.0 )
  3. Reviews ( Rel 2.0)
  4. Past Jobs,Feedback ( Rel 2.0)
  5. Travel distance, areas Served ( Rel 2.0)

2.6. Administration¶

  1. Approve a new tradie profile ( Rel 1.0 )
  2. Dispute resolution.(Rel 1.0)
  3. Complete logging of admin accounts. (Rel 1.0)
  4. edit of all fields except payments. (Rel 1.0)
  5. CustomerCare can create a job on behalf of user (Rel 2.0)

2.7. Payments¶

payments will be handled as

  1. [ Callfixie %] upfront [1] and rest of the amount between Tradie and User themselves. (Rel 1.0)
  2. Full amount upfront and refund in case of unsatisfactory work. (Rel 1.0).100 % deduction from user on job approval.

Footnotes

[1]for Hourly/continuous jobs this will be upfront amount decided by user.
